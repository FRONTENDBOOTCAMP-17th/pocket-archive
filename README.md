## ⚙️ 설치 및 실행 방법

```bash
# 1. 저장소 클론
git clone https://github.com/FRONTENDBOOTCAMP-17th/pocket-archive.git

# 2. 폴더 이동
cd pocket-archive

# 3. 의존성 설치 (npm 사용 시)
npm install

# 4. 프로젝트 실행
npm run dev
```
