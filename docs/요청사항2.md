# 📋 백엔드 API 수정 및 기능 추가 요청사항

### 1. 게시글 작성(Post) API Body 반환값 변경

- **대상 API:** `POST /api/pocket-archive/v1/posts`
- **변경 사항:** `body` 내에 **`preset` (Object)** 반환값 변경
- **수정된 요청 바디 (Request Body):**
  ```json
  {
    "title": "string",
    "content": "string",
    "category": "string",
    "preset": {
      "deckname": "string",
      "pocketmons": [1, 2, 100, 200, 224, 6],
      "gender": "women"
    },
    "imgUrl": "https://storage.fullstackfamily.com/.../image.webp"
  }
  ```

### 2. 게시글 수정(PUT) API Body 반환값 변경

- **대상 API:** `PUT /api/pocket-archive/v1/posts/${id}`
- **변경 사항:** `body` 내에 **`preset` (Object)** 반환값 변경
- **수정된 요청 바디 (Request Body):**

    "preset": {
